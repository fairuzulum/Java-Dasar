# ☕ Java-Dasar

Kumpulan latihan dan contoh program **Java dasar (fundamental)**. Repository ini berisi implementasi konsep-konsep inti Java — mulai dari tipe data, operator, percabangan, perulangan, hingga method — masing-masing dalam file terpisah agar mudah dipelajari satu per satu.

## 📚 Daftar Materi

| File | Konsep |
|------|--------|
| `Variable.java` | Deklarasi dan penggunaan variabel |
| `TipeDataPrimitive.java` | Tipe data primitif di Java |
| `ConversiTipeData.java` | Konversi antar tipe data (casting) |
| `Printing.java` | Output/print ke console |
| `UserInput.java` | Membaca input dari pengguna (Scanner) |
| `Aritmatika.java` | Operator aritmatika |
| `OperatorAssigment.java` | Operator assignment (`=`, `+=`, `-=`, dll) |
| `OperatorKomparasi.java` | Operator perbandingan |
| `OperatorLogika.java` | Operator logika (`&&`, `\|\|`, `!`) |
| `OperatorUnary.java` | Operator unary (`++`, `--`, dll) |
| `IfElseStatement.java` | Percabangan if-else |
| `SwitchCase.java` | Percabangan switch-case |
| `TernaryOperator.java` | Operator ternary |
| `ForLoop.java` | Perulangan for |
| `While.java` | Perulangan while |
| `DoWhile.java` | Perulangan do-while |
| `Method.java` | Deklarasi dan pemanggilan method |
| `Main.java` | Entry point utama |

## 🛠️ Tech Stack

- **Bahasa:** Java
- **IDE:** IntelliJ IDEA (`program.iml` disertakan)

## 🚀 Cara Menjalankan

### Menggunakan IntelliJ IDEA
1. Clone repository ini
   ```bash
   git clone https://github.com/fairuzulum/Java-Dasar.git
   ```
2. Buka folder project di IntelliJ IDEA
3. Jalankan file `.java` yang ingin dicoba (klik kanan → **Run**)

### Menggunakan Terminal (JDK)
```bash
cd src
javac NamaFile.java
java NamaFile
```
Contoh:
```bash
javac SwitchCase.java
java SwitchCase
```
